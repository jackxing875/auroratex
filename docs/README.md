# AuroraTeX Documentation

Welcome to the AuroraTeX documentation. AuroraTeX is a clean, modern, and
customizable LaTeX report template for academic reports, course projects, and
technical notes.

## Documentation Index

| Document | Description |
|---|---|
| [Getting Started](getting-started.md) | Quick start guide — install, compile, first document |
| [Customization](customization.md) | How to change colours, fonts, geometry, and styles |
| [Command Reference](commands.md) | All provided commands and environments |
| [Project Structure](structure.md) | Anatomy of the template directory |
| [Build & Compile](build.md) | Compiler setup, latexmk, Makefile, Overleaf |

## Quick Links

- **Class file (the core):** [`AuroraTeX.cls`](../AuroraTeX.cls) — contains all visual definitions and package loading.
- **Example main file:** [`main.tex`](../main.tex) — a fully-working example document.

## First Steps

If you are new to AuroraTeX, read the [Getting Started](getting-started.md)
guide.  It takes about 5 minutes and you will have a compiled PDF.
